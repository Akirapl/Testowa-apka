# Snake Nokia 3310

Klon klasycznej gry Snake z Nokii 3310 — dziala w przegladarce mobilnej i desktopowej.

## Uruchomienie

Otworz `index.html` w przegladarce. Nie wymaga serwera ani instalacji.

## Sterowanie

- **Mobilnie**: swipe po ekranie lub przyciski D-pad na ekranie
- **Desktop**: strzalki / WASD, Enter/Spacja aby rozpoczac
